## Description

These are JLC PCB specific files (files formatted and modified as per JLCPCB guidelines)
* JLCPCB BOM file
* JLCPCB CPL file

## How to use

1. Convert the gerber folder
**(pslab-mini-hardware/output/gerber)** to zip folder and upload it to the JLCPCB main page.
2. Turn the PCB assembly feature on(Rest all the options on this page can be kept as default).
3. On the Bill of materials page, add the BOM File and the CPL File provided in this folder.
4. On the next page all the components are directly selected using the JLCPCB part number present in the BOM file.
